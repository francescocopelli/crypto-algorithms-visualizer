# Crypto Algorithms Visualizer

Interactive educational web app for exploring cryptographic algorithms and their attacks.

## Features

- Step-by-step visualizations for major algorithms
- Attack Lab section to show why insecure constructions fail
- Student / Teacher mode
- Real numeric examples for DES, AES, RSA, and ECC
- Quiz mode for exam preparation
- Responsive UI

## Included algorithms

- DES
- AES
- RSA
- Diffie-Hellman
- ElGamal
- Hash functions
- MAC / HMAC / CMAC
- Authenticated Encryption
- ECC / ECDH
